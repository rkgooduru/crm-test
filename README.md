# CRM PowerApps Application

A comprehensive Microsoft PowerApps CRM system with role-based access control for managing Accounts, Contacts, Inventory, and User Administration.

## Project Overview

This project implements a cloud-based CRM solution using Microsoft PowerApps and Dataverse with the following modules:

### Core Modules
- **Accounts**: Customer/organization management
- **Contacts**: Individual contact management
- **Inventory**: Product/service tracking and management
- **User Admin**: Role-based user management
- **User Non-Admin**: Standard user interface with restricted permissions

## Documentation Structure

- `/specifications/` - Detailed module specifications
- `/dataverse/` - Dataverse schema and configuration
- `/powerapps/` - PowerApps app configurations
- `/workflows/` - Power Automate workflows
- `/security/` - Role and security configurations
- `/testing/` - Test cases and UAT documentation
